# Rahadigital (React + Vite)

Rahadigital is a modern, responsive digital agency website built with React and Vite. It includes a modular component architecture, dark mode, responsive navigation, and a scalable page structure inspired by leading full-service agencies.

## Tech Stack

- React, Vite, Tailwind CSS
- ESLint + Prettier
- Framer Motion for subtle animations

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- Images are intentionally omitted. Replace placeholders with your assets.
- Update content in `src/data/content.js` and related data files to customize sections.
- Page structure mirrors a multi-service agency site (Design, Digital Marketing, Technology, Solutions, Industries, Work, Insights, Consulting, Contact).
